📦 Inventory System (Складова наличност)

📌 Описание:
Това е прост Python проект за управление на складова наличност.
Позволява добавяне, премахване, търсене и анализ на продукти.

📁 Файлове:
- models.py -> съдържа клас Inventory с всички методи
- main.py -> демонстрация на функционалността
- requirements.txt -> библиотеки (няма външни зависимости)
- readme.txt -> описание на проекта

⚙️ Функции:
- Добавяне на продукт
- Премахване на продукт
- Търсене
- Обновяване на количество
- Сортиране по цена и количество
- Изчисляване на обща стойност
- Филтриране на ниска наличност
- Средна цена
- Брой продукти

▶️ Стартиране:
1. Инсталирай Python 3
2. Отвори терминал в папката
3. Стартирай:
   python main.py
